O que é Laços de repetição (loops)?

São uma das ferramentas mais poderosas para automatizar tarefas repetitivas no desenvolvimento de software.
Se você precisa processar uma lista de dados, realizar operações em múltiplos elementos de um array,
ou executar um conjunto de instruções várias vezes, os laços de repetição permitem que você faça isso de forma
eficiente e controlada. No contexto da programação, os laços são essenciais para melhorar a produtividade e reduzir a
complexidade do código.
Laços de repetição (loops) são blocos de código que executam várias vezes enquanto uma condição for verdadeira.
Eles são úteis para repetir tarefas, percorrer listas e automatizar processos.

                            Os principais tipos de loops são:


                                for: usado quando sabemos quantas vezes queremos repetir.

                                while: repete enquanto uma condição for verdadeira.

                                do-while: parecido com o while, mas garante que o bloco será executado pelo menos uma vez.

                                for-each: ideal para percorrer itens de uma lista ou coleção.
