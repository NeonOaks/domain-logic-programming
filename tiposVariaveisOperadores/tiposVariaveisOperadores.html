O que é uma variável na lógica de programação?

Uma variável na lógica de programação é um espaço na memória que armazena um valor que pode mudar durante a execução do programa. Ela funciona como uma caixa com um nome, onde você guarda um dado para usar depois.

Conceito Simples:

Variável = nome dado a um valor armazenado.

Exemplo do mundo real:

Imagine uma caixa com uma etiqueta:
Etiqueta: idade
Conteúdo da caixa: 25
Depois você pode mudar o valor dentro da caixa para 26.

Exemplo em pseudocódigo:

                                                variável idade = 25
                                                escreva(idade)  // Isso mostra 25
                                                idade = 26
                                                escreva(idade)  // Agora mostra 26
Por que variáveis são importantes?

Elas permitem que você:
- Guarde informações temporárias
- Reutilize dados
- Torne o programa mais flexível

Operadores Lógicos

    

        Os operadores lógicos são usados para comparar expressões booleanas (que resultam em True ou False) e combinar condições.
         Eles são essenciais para a tomada de decisão e estrutura de controle em programação, como em if, while e outras.
    

    

        Atribuição: Usada para definir ou atualizar o valor de variáveis. Além de = e +=, existem outros como -=, *=, /=, etc.

        Comparação: Compara valores e retorna true ou false. Prefira === e !== para evitar erros com conversão de tipos.

        Lógicos: Usados para combinar condições em estruturas como if. O || (OU) verifica se pelo menos uma condição é verdadeira,
        enquanto o && (E) exige que todas sejam verdadeiras.

        Diferentes: Incluí != e !== para cobrir o termo "diferentes". O !== é mais seguro por verificar o tipo.
