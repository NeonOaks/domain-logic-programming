O que é uma variavel na logica de programação ?

Uma variável na lógica de programação é um espaço na memória que armazena um valor que pode mudar durante a execução do programa. Ela funciona como uma caixa com um nome, onde você guarda um dado para usar depois

Conceito Simples:

Variável = nome dado a um valor armazenado.

Exemplo do mundo Real:

Imagine uma caixa com uma etiqueta: Etiqueta: idade Conteúdo da caixa: 25 Você pode depois mudar o valor dentro da caixa, por exemplo, para 26.

Exemplo em pseudocódigo:

variável idade = 25 escreva(idade) // Isso mostra 25 idade = 26 escreva(idade) // Agora mostra 26
Por que variáveis são importantes?
Elas permitem que você: Guarde informações temporárias Reutilize dados Torne o programa mais flexível
