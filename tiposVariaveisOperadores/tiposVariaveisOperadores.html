O que é uma variável na lógica de programação?

Uma variável na lógica de programação é um espaço na memória que armazena um valor que pode mudar durante a execução do programa. Ela funciona como uma caixa com um nome, onde você guarda um dado para usar depois.

Conceito Simples:

Variável = nome dado a um valor armazenado.

Exemplo do mundo real:

Imagine uma caixa com uma etiqueta:
Etiqueta: idade
Conteúdo da caixa: 25
Depois você pode mudar o valor dentro da caixa para 26.

Exemplo em pseudocódigo:

                                                variável idade = 25
                                                escreva(idade)  // Isso mostra 25
                                                idade = 26
                                                escreva(idade)  // Agora mostra 26
Por que variáveis são importantes?

Elas permitem que você:
- Guarde informações temporárias
- Reutilize dados
- Torne o programa mais flexível

Operadores Lógicos

        

            Os operadores lógicos e de comparação são ferramentas essenciais na programação. 
            Eles permitem comparar valores e combinar condições para tomar decisões em estruturas 
            como if, while e outras.
        


        
Conceito Simples:

        

            - Operadores de Atribuição: Atribuem valores a variáveis.

            - Operadores de Comparação: Comparam valores e retornam True ou False.

            - Operadores Lógicos: Combinam condições booleanas para decisões complexas.
        


        
Tipos de Operadores:

        

            Atribuição: Define ou atualiza valores de variáveis.

            Comparação: Verifica igualdade, diferença, maior/menor, etc.

            Lógicos: Combina condições com "E", "OU" ou inverte com "NÃO".
        


        
Exemplo em Pseudocódigo:

        
            variável idade = 20
            variável tem_carteira = Verdadeiro
            
            // Operadores lógicos
            se idade >= 18 e tem_carteira:
                escreva("Pode dirigir!")
            senão:
                escreva("Não pode dirigir!")
            
            // Operadores de comparação
            se idade == 20:
                escreva("Tem 20 anos!")
            
            // Operador lógico OU
            se idade < 18 ou tem_carteira:
                escreva("Pelo menos uma condição é verdadeira!")
        


        
Lista de Operadores:

        

            - Atribuição: =, +=, -=, *=, /=

            - Comparação: == (igual), != (diferente), > (maior), < (menor), >= (maior ou igual), <= (menor ou igual)

            - Lógicos: and (E), or (OU), not (NÃO)
        


        
Por que são importantes?

        

            Eles permitem:

            - Tomar decisões com base em condições

            - Controlar o fluxo do programa

            - Combinar múltiplas verificações para lógica complexa
