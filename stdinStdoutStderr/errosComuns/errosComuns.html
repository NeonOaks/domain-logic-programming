Erros Comuns em Programação

JavaScript

Erro de Sintaxe

Descrição: O código está malformado e não segue as regras da linguagem.

Exemplo no console:

Uncaught SyntaxError: Unexpected token ')'

Causa: Escrever console.log("teste"; (falta fechar o parêntese).

Erro de Tempo de Execução

Descrição: Ocorre durante a execução, quando o código tenta fazer algo inválido.

Exemplo no console:

Uncaught TypeError: Cannot read properties of null (reading 'value')

Causa: Tentar acessar document.getElementById("inexistente").value.

Erro Lógico

Descrição: O código executa, mas o resultado está errado devido a uma falha na lógica.

Exemplo no console:

Resultado: 10

Causa: Usar let soma = 5 + 5; quando a intenção era 5 * 5.

Erro de Tipo

Descrição: Operação inválida devido a tipos incompatíveis.

Exemplo no console:

Uncaught TypeError: num.toUpperCase is not a function

Causa: Tentar let num = 42; num.toUpperCase();.

Erro de Referência

Descrição: Tentativa de acessar uma variável ou função não definida.

Exemplo no console:

Uncaught ReferenceError: funcaoInexistente is not defined

Causa: Chamar funcaoInexistente(); sem defini-la.

Erro de Recursos

Descrição: Falha ao acessar recursos externos, como rede ou arquivos.

Exemplo no console:

POST http://127.0.0.1:5500/executar 405 (Method Not Allowed)

Causa: Enviar fetch("/executar", { method: "POST" }) para um servidor que não suporta POST.

Java

Erro de Sintaxe

Descrição: Código não segue a sintaxe do Java.

Exemplo no terminal:

Main.java:3: error: ';' expected System.out.println("Hello" ^

Causa: Escrever System.out.println("Hello" sem fechar com );.

Erro de Tempo de Execução

Descrição: Erro durante a execução, como acessar algo inválido.

Exemplo no terminal:

Exception in thread "main" java.lang.NullPointerException at Main.main(Main.java:5)

Causa: Tentar String s = null; s.length();.

Erro Lógico

Descrição: O programa roda, mas o resultado está errado.

Exemplo no terminal:

Resultado: 15

Causa: Usar int result = 10 + 5; quando a intenção era 10 * 5.

Erro de Tipo

Descrição: Tipos incompatíveis em uma operação.

Exemplo no terminal:

Main.java:5: error: incompatible types: String cannot be converted to int int num = "123abc"; ^

Causa: Atribuir uma string a uma variável int.

Erro de Referência

Descrição: Acessar uma variável ou método inexistente.

Exemplo no terminal:

Main.java:5: error: cannot find symbol inexistente(); ^

Causa: Chamar inexistente() que não foi definido.

Erro de Recursos

Descrição: Falha ao acessar recursos como arquivos ou rede.

Exemplo no terminal:

Exception in thread "main" java.io.FileNotFoundException: arquivo.txt (No such file or directory) at java.io.FileInputStream.open0(Native Method)

Causa: Tentar abrir new FileInputStream("arquivo.txt").

TypeScript

Erro de Sintaxe

Descrição: Código malformado que viola as regras do TypeScript.

Exemplo no terminal:

app.ts:2:1 - error TS1005: ';' expected. console.log("teste"

Causa: Escrever console.log("teste" sem fechar com );.

Erro de Tempo de Execução

Descrição: Erro ao executar o código compilado.

Exemplo no console:

Uncaught TypeError: Cannot read properties of undefined (reading 'length')

Causa: Acessar let arr: string[] | undefined; arr.length; sem inicializar arr.

Erro Lógico

Descrição: Código executa, mas o resultado está incorreto.

Exemplo no console:

Resultado: 8

Causa: Usar let result: number = 4 + 4; quando a intenção era 4 * 4.

Erro de Tipo

Descrição: Violação das regras de tipagem do TypeScript.

Exemplo no terminal:

app.ts:3:5 - error TS2322: Type 'string' is not assignable to type 'number'. let num: number = "123";

Causa: Atribuir uma string a uma variável tipada como number.

Erro de Referência

Descrição: Referenciar algo não definido.

Exemplo no terminal:

app.ts:4:5 - error TS2304: Cannot find name 'funcaoInexistente'. funcaoInexistente();

Causa: Chamar funcaoInexistente() sem defini-la.

Erro de Recursos

Descrição: Falha ao acessar recursos externos.

Exemplo no console:

POST http://127.0.0.1:5500/executar 405 (Method Not Allowed)

Causa: Enviar fetch("/executar", { method: "POST" }) para um servidor que não suporta POST.

SQL

Erro de Sintaxe

Descrição: Consulta SQL malformada.

Exemplo no terminal:

ERROR: syntax error at or near "FORM" at character 16 SELECT * FORM usuarios;

Causa: Escrever FORM em vez de FROM.

Erro de Tempo de Execução

Descrição: Erro durante a execução da consulta.

Exemplo no terminal:

ERROR: relation "tabela_inexistente" does not exist

Causa: Tentar SELECT * FROM tabela_inexistente;.

Erro Lógico

Descrição: Consulta executa, mas retorna dados errados.

Exemplo no terminal:

(Retorna 0 linhas)

Causa: Usar SELECT * FROM usuarios WHERE idade > 30; quando a intenção era idade < 30.

Erro de Tipo

Descrição: Operação com tipos incompatíveis.

Exemplo no terminal:

ERROR: operator does not exist: text + integer

Causa: Tentar SELECT nome + idade FROM usuarios;.

Erro de Referência

Descrição: Referenciar uma coluna ou tabela inexistente.

Exemplo no terminal:

ERROR: column "coluna_inexistente" does not exist

Causa: Tentar SELECT coluna_inexistente FROM usuarios;.

Erro de Recursos

Descrição: Falha ao acessar recursos do banco.

Exemplo no terminal:

ERROR: could not connect to server: Connection refused

Causa: Tentar conectar a um banco de dados inativo.
