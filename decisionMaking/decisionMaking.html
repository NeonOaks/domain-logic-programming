Estruturas Condicionais:

Estruturas Condicionais, como if, else if e else, permitemque o programa tome decisões com base em condições verdadeiras ou falsas.

Conceito Simples:

O if Verifica uma condição. Se for verdadeira, executa um bloco de código. O else if testa condições adicionais, e o else executa um bloco padrão se nenhuma condições for verdadeira.

Exemplo em Código:

                        let idade = 16;
                        if(idade >= 18) {
                            console.log("Maior de idade!");
                        } else if (idade >=16) {
                            console.log("Quase maior de idade!");
                        } else {
                            console.log("Menor de idade!");
                        }
                        //Saida: Quase Maior de idade!
Por que são importantes?

Elas permitem:
- Tomar decisções com base em condições
- Controlar o fluxo de programa
- Criar lógica dinâmica
