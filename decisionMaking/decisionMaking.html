Estruturas Condicionais:

Estruturas Condicionais, como if, else if e else, permitemque o programa tome decisões com base em condições verdadeiras ou falsas.

Conceito Simples:

O if Verifica uma condição. Se for verdadeira, executa um bloco de código. O else if testa condições adicionais, e o else executa um bloco padrão se nenhuma condições for verdadeira.

Exemplo em Código:

                        let idade = 16;
                        if(idade >= 18) {
                            console.log("Maior de idade!");
                        } else if (idade >=16) {
                            console.log("Quase maior de idade!");
                        } else {
                            console.log("Menor de idade!");
                        }
                        //Saida: Quase Maior de idade!
Por que são importantes?

Elas permitem:
- Tomar decisções com base em condições
- Controlar o fluxo de programa
- Criar lógica dinâmica

Estrutura Switch Case

        

            A estrutura switch case é usada para executar diferentes blocos de código com base no valor de uma expressão. É uma alternativa mais limpa a múltiplos if else.
        


        
Conceito Simples:

        

            O switch compara uma expressão com valores definidos em case. Se houver correspondência, executa o código do caso. O default lida com valores não previstos.
        


        
Exemplo em Código:

        
                    let dia = 2;
                    switch (dia) {
                        case 1:
                            console.log("Segunda-feira");
                            break;
                        case 2:
                            console.log("Terça-feira");
                            break;
                        case 3:
                            console.log("Quarta-feira");
                            break;
                        default:
                            console.log("Dia inválido");
                    }
                    // Saída: Terça-feira
        


        
Por que é importante?

        

            Permite:

            - Simplificar múltiplas condições

            - Tornar o código mais legível

            - Lidar com seleções baseadas em valores fixos
